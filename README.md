# VR UI Lab

A lab to experiment with interactions in VR

Technologies used
* Unity 3d
* Oculus OVR controller
* Unity XR Interaction Toolkit

Demo Video (Right click image to open in new tab)

Packages to import
* Fantasy Skybox
* Ingame Debug Console
* Oculus Integration
* XR Interaction Toolkit
* TextMesh Pro
* Voxel Animals
